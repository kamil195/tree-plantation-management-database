# Tree Plantation Management Database

A MySQL database project for recording trees, volunteers, and plantation activities. It demonstrates relational database design, joins, views, aggregate queries, subqueries, triggers, transactions, and indexing.

## Database entities

- Trees
- Volunteers
- Plantation activities

## Repository structure

- `database/tree_plantation_database.sql`
- `docs/tree_plantation_database_report.docx`

## Running the project

Open MySQL Workbench or another MySQL client and run `database/tree_plantation_database.sql`.
